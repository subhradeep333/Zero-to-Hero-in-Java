<h2>📘 About This Repository</h2>
<p>
This repository contains <b>Java practice questions with answers</b> for beginners.
The repository is organized <b>topic-wise (folder-wise)</b> so that students can easily
practice questions from Arrays, Strings, Patterns, Numbers, Sorting, Searching, and OOP.
This repository is mainly created for students who are starting Java and want to practice programming.
</p>
